Ac1dRa1n

Custom-length custom-model random password generator.

Generates tough and purely randomized passwords.

Getting Ruby

OS X Users :

$ brew install ruby

Debian/GNU/Linux :

$ sudo apt-get install ruby-full

Running Ac1dRa1n

Run this command from the HackSaw directory :

$ ruby script.rb

Note : Ensure all require_relative files are in the same directory as script.rb
